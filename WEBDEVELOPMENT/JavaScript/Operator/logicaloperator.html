Logical Operator

Logical AND (&&) , OR (||) , NOT (!) operators

  1. Logical AND (&&) operator

    2. logical "AND" operator in JavaScript, represented by "&&", evaluates two operands.
    If both operands are true, it returns true.
    Otherwise, it returns false.



console.log(true && true); // true
console.log(false && true); // false
console.log(10 > 0 && 10 < 50); // true

Logical OR (||) operator

It returns true if any operand is true and false only if all operands are false.


console.log(false || false); // false
console.log(false || true); // true
console.log(10 < 0 || 10 > 50); // false

Logical NOT (!) operator

Whenever there is a need to inverse the boolean value, the logical NOT (!) operator is used.

console.log(!true); // false console.log(!10); // false console.log(!0); // true
