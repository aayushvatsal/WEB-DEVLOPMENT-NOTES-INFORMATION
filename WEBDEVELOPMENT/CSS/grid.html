Introduction to CSS Grid

CSS Grid Layout is a two-dimensional grid-based layout system. Grid Layout Module offers a grid-based layout system, with rows and columns, making it easier to design web pages without having to use floats and positioning.


Grid Elements

A grid layout consists of a parent element, with one or more child elements.

A Grid Layout must have a parent element with the display property set to
grid or inline-grid.

Direct child element(s) of the grid container automatically becomes grid items.

Display Property

An HTML element becomes a grid container when its display property is set to grid or inline-grid.

display : grid


syntax :


.grid-container {
display: grid;
}

display : inline-grid


syntax :


.grid-container {
display: inline-grid;
}

Grid container

To make an HTML element behave as a grid container, we have to set the display property to grid or inline-grid.

The grid-template-columns Property

  • The grid-template-columns property defines the number of columns in your grid layout, and it can define the width of each column.
  • The value is a space-separated-list, where each value defines the width of the respective column.

  • The value is a space-separated-list, where each value defines the width of the respective column.

CSS Grid Item

A grid container contains grid items.
By default, a container has one grid item for each column, in each row, but you can style the grid items so that they will span multiple columns and/or rows.

The grid-column Property:

The grid-column property defines on which column(s) to place an item. we define where the item will start, and where the item will end.

Item1 will start on column 1 and end before column 5:


syntax :


.grid-container {
grid-column: 1 / 5;
}

Naming Grid Items

The grid-area property can also be used to assign names to grid items.

Grid Layout : 2-dimension

Header
body
sidebar
footer

    .page{ 

      display: grid; 

      grid-template-areas:"nav-header nav-header nav-header panel-sidebar" 

      "p-body p-body  p-body panel-sidebar" 

      "b-footer b-footer b-footer b-footer" ; 

   } 


   .page-header{ 

    grid-area: nav-header; 

    background-color: red; 

    padding: 15px; 

  }
  
  .page-body{ 

     grid-area: p-body; 

     background-color: orange; 

     padding: 15px; 

  } 

  
  .page-sidebar{ 

     grid-area: panel-sidebar; 

     background-color: brown; 

     padding: 15px; 

  } 

  
  .page-footer{ 

     grid-area: b-footer; 

     background-color: teal; 

     padding: 15px;

  }
