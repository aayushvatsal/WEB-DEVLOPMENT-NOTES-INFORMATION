CSS Selector

CSS selectors

are used to select the content we want to style. Selectors are the part of CSS rule set.
CSS selectors select HTML elements according to its id, class, type, attribute etc.

There are several different types of selectors in CSS.

  1. CSS Element Selector
  2. CSS Id Selector
  3. CSS Class Selector
  4. CSS Universal Selector
  5. CSS Group Selector

CSS Element Selector

The element selector selects the HTML element by name.

Syntax :


a {
text-align: center;
color: red;
}

Here, all 
 < a >
elements on the page will be center-aligned, with a red text color:

CSS Id Selector :

The id selector selects the id attribute of an HTML element to select a specific element.
An id is always unique within the page so it is chosen to select a single, unique element.

It is written with the hash character (#), followed by the id of the element.

syntax :


#para1 {
text-align: center;
color: green;
}

The CSS rule will be applied to the HTML element with id="para1":

CSS Class Selector :

The class selector selects HTML elements with a specific class attribute.

It is used with a period character . (full stop symbol) followed by the class name.

syntax :


.box1 {
text-align: center;
color: green;
}

CSS Universal Selector

The universal selector is used as a wildcard character. It selects all the elements on the pages.

syntax :


*{
text-align: center;
color: green;
}

CSS Group Selector

The grouping selector is used to select all the elements with the same style definitions.


Grouping selector is used to minimize the code.

Commas are used to separate each selector in grouping.

syntax :


h1 {
text-align: center;
color: red;
}
h2 {
text-align: center;
color: red;
}
p {
text-align: center;
color: red;
}

It will be better to group the selectors, to minimize the code.
To group selectors, separate each selector with a comma.

syntax :


h1, h2, p{
text-align: center;
color: green;
}

CSS Combinators

A combinator is something that explains the relationship between the selectors.

There are four different combinators in CSS:

  • descendant selector (space)
  • child selector (>)
  • adjacent sibling selector (+)
  • general sibling selector (~)

Descedent Selector

The Descendant combinator is represented using a single space.
It combines two selectors in which the first selector represents an ancestor (parent, parent's parent, etc.), and the second selector represents descendants.

syntax :


first_selector second_selector {
css properties: values;
}

  • A first simple selector (first_selector) represents an ancestor element,
    while a second simple selector (second_selector) represents a descendant element.

Let us understand the above syntax representation with the following example:

syntax :


h2 p {
background-color: blue;
font-size: 2rem;
}

  • Here, h2 acts as the ancestor element while p is the descendant element.
  • The white space between the two elements represents the descendant selector.

Child Selector

The child selector selects all elements that are the children of a specified element.
The following example selects all

elements that are children of a

element:

syntax :


div > p {
background-color: powderblue;
}

Adjacent Sibling Selector

An adjacent sibling combinator selector allows us to select an element that is directly after another specific element.


it selects those elements that are nested under the same parent as a specified element and also appear immediately after that element.

The adjacent sibling selector is represented by a plus (+) sign in between two selectors.
The second selector represents the element that is to be selected when it appears immediately after an element represented by the first selector,
provided both the elements are children of the same parent.

syntax :


h1 + p {
background-color: powderblue;
}

General sibling selector

The general sibling selector it selects those elements in a document that are nested under the same parent element as a specified element and is also present after that element.

syntax :


h1 ~ p {
background-color: powderblue;
}

Pseudo Class Selector

pseudo-class selector is a keyword added to a selector that specifies a special state of the selected elements.

It can be combined with a CSS selector to add an effect to existing elements based on their states.

Here are some common pseudo-classes in CSS:

1. :hover -

Selects and styles an element when the user hovers over it.

syntax :


a:hover {
color: red;
}

2. :active -

Selects and styles an element while it is being activated (e.g., clicked).

syntax :


button:active {
background-color: #00ff00;
}

3. :focus -

Selects and styles an element when it has focus.

syntax :


input:focus {
border: 2px solid blue;
}

4. :link -

when the link has not yet been clicked.

syntax :


a:link {
color: green;
}

5. :nth-child() -

Selects and styles elements based on their position within a parent.

syntax :


li:nth-child(odd) {
background-color: #f0f0f0;
}

6. :first-child, :last-child -

Selects the first or last child of a parent element.

syntax :


li:first-child {
font-weight: bold;
}
li:last-child {
color: red;
}

7. :nth-of-type() -

Selects and styles elements based on their position among siblings of the same type.

syntax :


p:nth-of-type(2n) {
color: #3366cc;
}

Pseudo Element Selector

A CSS pseudo-element is used to style specified parts of an element.

We have used the double colon notation (::pseudo-element) in the syntax.

The ::first-letter Pseudo-element

The ::first-letter pseudo-element is used to add a special style to the first letter of a text.

It affects the first letter of the text. It can be applied only to block-level elements.

syntax :


p::first-letter {
color: #ff0000;
font-variant: small-caps;
}
