What is Align-Item?

The align-items property specifies the default alignment for items inside a flexbox

  • In a flexbox container, the flexbox items are aligned on the cross axis,
    which is vertical by default (opposite of flex-direction).

The Cross-Axis Axis

The cross axis is defined by flex-direction

  • Align-Item : Flex-Start
  • Align-Item : Flex-end
  • Align-Item : Flex-Center
  • Align-Item : Flex-baseline
  • Align-Item : Flex-Stretch(By default)
