JavaScript Ternary Operator

Ternary Operator in JavaScript is a special operator which has three operands.
In JavaScript, there is only one such operator i.e. Conditional Operator ( ?: ).

Ternary Operator Syntax

syntax :
condition ? expression_if_true : expression_if_false

The ternary operator is basically a combination of a question mark and a colon.
To use this operator, we need to write a conditional expression, followed by a question mark (?)
Then we need to provide two statements or expressions separated by a colon.
The first expression ( expression_if_true ) evaluates when the condition results in a true value,
and the second expression ( expression_if_false ) evaluates when the condition results in a false value.

Example 1. Suppose we want to check if a person passed or failed an examination.
This is how the if-else block would look like:

var marks = 50;
if(marks >= 40)
{
console.log("Passed");
}
else {
console.log("Failed");
}

Using the Ternary Operator, we can write the above block in just one line.

console.log(( marks >= 40 ) ? "Passed" : "Failed" );

In both cases, the output is:

Output : Passed

Why Use the Ternary Operator in JavaScript?

It is used to make our code short and simpler.
