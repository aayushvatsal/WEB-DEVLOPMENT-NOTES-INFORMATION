List In Html

List are used to represent real life list data.

Unordered HTML List

An unordered list starts with the 
 < ul > tag.
Each list item starts with the 
< li >tag
.
The list items will be marked with bullets (small black circles) by default:
  • Coffee
  • Tea
  • Milk
  • Biscuit
  • Snacks
            syntax : < ul >
                < li>Coffee< /li>
                < li>Tea< /li>
                < li>Milk< /li>
                < li>Biscuit< /li>
                < li>Snacks< /li>
              < / ul >

Unordered HTML List - Choose List Item Marker

HTML Ordered Lists

The HTML 
 < ol> tag defines an ordered list.
An ordered list can be numerical or alphabetical.
We use the HTML ordered list to define a list where the sequence or order of the list items is important.
We can use the HTML ordered list for recipes, algorithms, top ten lists, and so on.
  1. Coffee
  2. Tea
  3. Milk
  4. Biscuit
  5. Snacks
        syntax : < ol >
            < li> Coffee < /li>
            < li>Tea < /li>
            < li>Milk < /li>
            < li>Biscuit < /li>
            < li>Snacks < /li>
          < / ol >

Ordered HTML List - Choose List Item Marker

Control List Counting

By default, an ordered list will start counting from 1.
If you want to start counting from a specified number, you can use the start attribute:

Example

  1. Coffee
  2. Tea
  3. Milk
  4. Biscuit
  5. Snacks
        syntax : < ol start="20" >
            < li>Coffee< /li>
            < li>Tea< /li>
            < li>Milk< /li>
            < li>Biscuit< /li>
            < li>Snacks< /li>
            < / ol >

HTML Description Lists

HTML also supports description lists.
A description list is a list of terms, with a description of each term. 

        
        The < dl> tag defines the description list, the < dt> tag defines the term (name), and the < dd> tag describes each term:

Example

Juice
- Banana Juice
- Mango Juice
- Apple Juice
Cold Drink
- Pepsi
- Coca-cola
- Maza
            Syntax :
            < dl>
            < dt>Juice< /dt>
            < dd>- Banana Juice < /dd>
            < dd>- Mango Juice < /dd>
            < dd>- Apple Juice < /dd>
            < dt>Cold Drink < /dt>
            < dd>- Pepsi < /dd>
            < dd>- Coca-cola < /dd>
            < dd>- Maza < /dd>
          < /dl>
